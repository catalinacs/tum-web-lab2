# Cookie Nest — Landing Page

A beautiful, fully-responsive landing page for **Cookie Nest**, a family-run artisanal cookie bakery. This project showcases 8 specialty cookie varieties with an interactive ordering system and elegant product showcase.

## Overview

Cookie Nest is a modern e-commerce landing page built entirely with **vanilla HTML, CSS, and JavaScript**. The site features:

- **Hero Section** with call-to-action buttons
- **Interactive Pricing System** where customers can select their desired package (Single, Half Dozen, Dozen, Party Platter) with real-time selection feedback
- **Detailed Cookie Menu** displaying 8 unique cookie varieties with images and descriptions
- **About Section** explaining the bakery's philosophy
- **Contact Form** for customer inquiries
- **Fully Responsive Design** optimized for mobile, tablet, and desktop
